## 🚀 Sobre Mí

Soy **Frontend Developer** con enfoque en crear interfaces limpias, escalables y centradas en el usuario. Transformo problemas complejos en experiencias web intuitivas y de alto rendimiento.

- 💼 **Especialización:** Aplicaciones web con HTML, CSS y JavaScript, diseño UI/UX con Figma, y desarrollo de componentes con React y Tailwind CSS.
- ⚡ **Filosofía de trabajo:** Código limpio, buenas prácticas y atención al detalle en cada pixel.
- 🌱 **Aprendiendo actualmente:** React avanzado, arquitecturas de componentes y optimización de rendimiento web.
- 🎓 **Universidad Politécnica de Chiapas** — Ingeniería en Desarrollo de Software.
